# CHANGELOG

## 1.2.0 (2026-03-13)

* Added sequences, sequence values and move-sequence, clear-sequence, add-to-sequence and remove-from-sequence actions.
* Instead of limiting 'and' and 'or' ifs to exactly two child conditionals, don't put an upper limit on the number of child conditionals, as long as there are at least two.
* Corrected the oneOf exclusion pattern of multiple start/end action properties in 'action-full'.

## 1.1.0 (2025-06-27)

* Initial version of the TimeSeq Script 1.1.0 JSON Schema definition.
